Secrets and Model Config

Date: 2026-05-27

Rule

API keys are local secrets. They must never be committed to git, written into HTML docs, copied into generated run artifacts, or logged through observability.

Local Setup

  1. Copy .env.example to a local .env file.
  2. Set OPENAI_API_KEY in .env.
  3. Set model names explicitly only when a model-assisted step is introduced.

Model Choice

The project does not currently choose OpenAI models automatically. This is intentional. The old project spent too much money partly because model choice was not disciplined. Future model calls must document why a model was chosen, what the task risk is, and whether the call is needed at all.

Current Implementation

  1. .env and .env.* are ignored by git.
  2. .env.example is committed as a safe template.
  3. src/config.py can load local environment values without adding a dependency.
  4. OPENAI_MODEL_FAST and OPENAI_MODEL_STRONG must be explicit before use.
