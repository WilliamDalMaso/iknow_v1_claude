Operating Position

We are starting fresh. The older project is a source of evidence, not a template. We will borrow useful disciplines such as evaluation, provenance, bounded experiments, and durable history. We will not inherit its assumption that raw PDF text plus chunking is good enough as the first substrate.

Initial Build Constraint

Work begins with one book only. The first engine milestone is not a chatbot, graph, or UI. It is a verified extraction package: every page accounted for, every content object typed, cleaned text separated from raw text, and all omissions/noise decisions recorded.
