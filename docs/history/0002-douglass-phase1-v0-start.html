Douglass Phase 1 v0 Start

Date: 2026-05-27

Book Choice

The first book is douglass_narrative.pdf. It was chosen over the_hobbit.pdf because it is smaller, better suited to public-domain experimentation, and already exposes real extraction issues without forcing v0 to solve a larger illustrated copyrighted book first.

What Was Built

Phase 1 v0 now has a deterministic extractor at src/phase1_extract.py and a formal artifact contract at docs/strategy/0002-phase-1-artifact-contract.html.

What The Run Exposed

  1. Page 1 contains visible (cid:...) extraction noise in the title/front-matter area.
  2. Page 2 has no extracted text and one image, so it needs image/OCR-aware review later.
  3. The v0 object model is line-based. That is useful for auditability but not yet a clean paragraph/object model.
  4. Heading detection is intentionally provisional and likely over-detects short uppercase lines.
This is the right failure shape for v0: we have a complete page inventory and visible risks, not a silent claim that extraction is clean.

Next Improvement

The next Phase 1 slice should improve object segmentation. Specifically, it should merge normal prose lines into paragraph objects while preserving headings, page artifacts, image-only pages, and flagged noisy objects separately.
