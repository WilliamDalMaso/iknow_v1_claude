Cross-Page Join Review

Date: 2026-05-27

Purpose

The v2_cross_page_continuation experiment improved the authoritative gold score, but it proposed 79 joins. This checkpoint adds a focused review layer so those side effects can be inspected before the policy becomes active.

This is review-only. The active canonical extraction policy remains v1_consecutive_lines; the experimental policy remains v2_cross_page_continuation. No OCR, AI review, embeddings, retrieval, graph work, user-facing answers, or structure promotion was added.

Generated Artifact

The run now generates cross_page_join_review_report.json. Each row records the left and right page, candidate ids, text previews, continuation evidence, gold coverage, risk category, confidence, recommended action, and audit anchors.

Douglass Result

  1. Total proposed joins: 79.
  2. Likely correct joins: 48.
  3. Possible false joins: 0.
  4. Boundary or structure risk joins: 4.
  5. Needs manual review: 27.
  6. Covered by authoritative gold: 6.
  7. Not covered by gold: 73.
The gold-covered joins explain why the experiment scored well, but most proposed joins are not covered by the current gold set. Adoption still requires reviewing the unscored joins, especially early/front-matter joins and unusually long joined paragraphs.

Audit Update

phase1_audit.html now includes a Cross-Page Join Review section with summary counts, top pages needing review, row-level risk categories, gold coverage, continuation evidence, text previews, and links back to the page/object audit anchors.

Crack Fixed

During regeneration, the code exposed an inconsistency: the cross-page policy was named as active even though canonical outputs were still built from the baseline evaluation. The code now records the active policy separately from the experimental policy so future runs do not confuse experiment evidence with adopted behavior.

Next Step

Inspect the 73 joins not covered by authoritative gold before adopting v2_cross_page_continuation. A small next review set should focus on the 4 boundary/structure risks and the 27 joins marked needs_manual_review.
