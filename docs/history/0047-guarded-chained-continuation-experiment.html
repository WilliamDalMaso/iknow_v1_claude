Guarded Chained Continuation Experiment

Date: 2026-05-28

Purpose

The previous chained-continuation side-effect review found a real false join: chained_join_review_0004 on pages 59-61. The unguarded v3_chained_cross_page_continuation policy skipped intervening poem/prose content on page 60. This checkpoint tested a tightened experiment-only policy: v3_chained_cross_page_continuation_guarded.

This checkpoint did not adopt the guarded policy, did not make it active, did not change canonical promotion rules, and did not add OCR, model review, retrieval, embeddings, graph work, user-facing answers, or structure promotion.

Guard Rule

A chained continuation must not skip later non-furniture paragraph content on the terminal page of the existing cross-page joined candidate. If such body content exists before the proposed next-page candidate, the chain is rejected as a side-effect risk.
