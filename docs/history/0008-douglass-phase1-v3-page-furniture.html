Douglass Phase 1 v3 Page Furniture

Date: 2026-05-27

Purpose

V3 tested whether deterministic evidence can separate repeated page furniture from real book structure. The trigger was the v2 result: only one page artifact candidate across 126 pages, while the audit showed many repeated running headers.

Page furniture is excluded only as a candidate. No source content is deleted, and every artifact remains inspectable in phase1_audit.html.

Method

  1. Normalize object text.
  2. Strip leading or trailing page-number-like tokens before repetition checks.
  3. Require short text near a page margin.
  4. Require the normalized furniture text to repeat across pages.
  5. Require repeated nearby vertical position for strong running-header candidates.
  6. Record artifact subtype, confidence, warnings, classification reasons, and furniture evidence.

Risks

The detector is intentionally deterministic, but it still needs human audit before promotion.
  1. Some front-matter running headers such as PREFACE vii are now artifacts, which is probably correct but should be inspected.
  2. Repeated section labels could be misclassified if they appear near a margin with attached page numbers.
  3. Page images are still not shown beside objects, so visual confirmation is not complete.
