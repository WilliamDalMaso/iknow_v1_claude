Cross-Page Join Decisions

Date: 2026-05-28

Purpose

The cross-page continuation experiment proposed joins outside the authoritative gold set. This checkpoint adds a durable decision workflow so reviewed join decisions can be tracked under reviews/, replayed into generated runs, and validated before adoption.

This is review governance only. The active canonical extraction policy remains v1_consecutive_lines; v2_cross_page_continuation remains experimental. No OCR, AI review, embeddings, retrieval, graph work, user-facing answers, or structure promotion was added.

Tracked Review File

The tracked source is reviews/douglass_narrative/cross_page_join_decisions.jsonl. Rows can mark proposed joins as accept, reject, or needs_review. The generated run replays those rows into cross_page_join_decisions_applied.jsonl.

Validation Rules

  1. Decision rows must include the join id, pages, candidate ids, decision, reason, reviewer, date, and evidence reference.
  2. Decision values must be accept, reject, or needs_review.
  3. Duplicate decisions for the same join are rejected.
  4. Decision rows must reference existing proposed join ids.
  5. The page and candidate ids in a decision row must match the proposed join.
