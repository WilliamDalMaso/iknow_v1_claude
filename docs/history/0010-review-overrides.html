Review Overrides

Date: 2026-05-27

Purpose

The audit can now reveal likely false positives. This step adds the persistent correction layer: review_overrides.jsonl. It lets a reviewer correct a candidate bucket without changing detector code and without promoting content to canonical.

Overrides are candidate-level corrections. They preserve the original bucket and add reviewer evidence; they do not create canonical book content.

Format

{
  "object_id": "douglass_narrative:p0012:obj003",
  "corrected_bucket": "structure_candidate",
  "reason": "This is a real preface heading, not page furniture.",
  "reviewer": "human",
  "date": "2026-05-27",
  "evidence_reference": "phase1_audit page 12 object obj_012_003"
}
