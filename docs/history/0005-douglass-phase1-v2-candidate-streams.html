Douglass Phase 1 v2 Candidate Streams

Date: 2026-05-27

Trial Purpose

This trial tested the candidate-first reconstruction idea on the Douglass PDF only. The goal was not perfect extraction. The goal was to make every extracted object visible in exactly one bucket so we can audit whether the separation is trustworthy page by page.

No Phase 1 output is canonical yet. The v2 run produces candidates for audit and later promotion.

Method

  1. Preserve page-level raw text in raw_pages.jsonl.
  2. Build deterministic layout objects from extracted text lines.
  3. Merge likely body prose lines into paragraph-shaped objects using line order and indentation.
  4. Classify each object into one candidate bucket: paragraph, structure, page artifact, or unknown.
  5. Write a reconstruction map candidate linking streams back to source objects, lines, pages, and available bounding boxes.
  6. Render phase1_audit.html with counts, validation status, flagged pages, and stream samples.

What Went Right

  1. The run no longer labels the provisional reading order as canonical.
  2. Every candidate row keeps source_object_ids and source_line_ids.
  3. Paragraph objects now carry a simple combined bounding box when line geometry is available.
  4. The validator checks that candidate stream objects exactly match layout objects.
  5. The audit page exposes candidate stream counts and samples.

What Went Wrong Or Is Still Weak

V2 proves the candidate-stream architecture, not extraction cleanliness.
  1. Only one page artifact was detected, which means repeated headers and page furniture are probably under-classified.
  2. Structure detection is still a simple deterministic heuristic. It does not yet distinguish chapter heading, title, running header, caption, or footnote.
  3. The audit page has samples, but it does not yet show page-by-page original object evidence beside bucket assignments.
  4. No OCR or page-image witness has been added yet.
  5. No model has reviewed ambiguous candidates yet, which is intentional for this trial.
