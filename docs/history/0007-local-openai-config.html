Decision

The project may use OPENAI_API_KEY locally, but the key must never be committed, logged, or written into generated artifacts.

Model Discipline

Model names are not defaulted. Future model-assisted work must choose a model explicitly and document why that model is appropriate for the task. This protects the project from repeating the old system's expensive trial-and-error pattern.
